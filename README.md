# SheafUI Components

Browse and copy SheafUI Blade components directly to your clipboard from Raycast.

## Features

- **Search Components**: Quickly search through all 35+ SheafUI components
- **Quick Copy**: Copy component blade files directly to your clipboard
- **Preview**: View component source code before copying
- **Documentation**: Quick access to SheafUI documentation
- **GitHub**: Direct links to component source on GitHub

## Usage

1. Open Raycast
2. Type "Browse SheafUI Components"
3. Search for a component
4. Actions:
   - `Enter` - View component files
   - `Cmd + C` - Quick copy all blade files
   - `Cmd + O` - Open documentation
   - `Cmd + Shift + O` - Open GitHub source
   - `Cmd + Shift + C` - Copy artisan install command

## Component Detail View

When viewing a component:
- See all blade files
- Preview file contents
- Copy individual files with `Cmd + C`
- Copy all files with `Cmd + Shift + C`
- Use `Cmd + 1-9` to switch between files

## Available Components

The extension includes all SheafUI components:

Accordion, Alert, Autocomplete, Avatar, Badge, Brand, Breadcrumbs, Button, Card, Checkbox, Description, Dropdown Menu, Error Message, Field, Fieldset, Heading, Icon, Input Field, Key Value Pair, Label, Link, Modal Dialog, OTP Input, Popover, Radio Group, Select, Separator, Slider, Switch Toggle, Tabs, Tags Input, Text, Textarea, Theme Switcher, Toast Notification, Tooltip
